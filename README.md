# 🍔 Fast Food Image Classification باستخدام CNN

This project focuses on building a Convolutional Neural Network (CNN) model to classify fast food images into 10 categories.

---

## 📌 Project Overview
The goal of this project is to apply deep learning techniques to solve an image classification problem using TensorFlow and OpenCV.

The model classifies images into:
- Burger
- Pizza
- Fries
- Donut
- Sandwich
- Taco
- Hot Dog
- Crispy Chicken
- Taquito
- Baked Potato

---

## 🧠 Model Architecture
- 3 Convolutional Layers (Conv2D)
- MaxPooling Layers
- Fully Connected Layers (Dense)
- Dropout for regularization
- Softmax output layer (10 classes)

---

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Features
- Image preprocessing and normalization
- Train/Validation/Test split
- Model training with EarlyStopping
- Performance evaluation (Accuracy, Confusion Matrix, Classification Report)

---

## 📈 Results
The model was able to classify fast food images with good performance using CNN architecture and proper preprocessing techniques.
